# Команды, которые фронт шлет на сервер 

---

## 1. **Управление поливом**

```json
{
  "action": "water",
  "data": {
    "state": "1"
  }
}

{
  "action": "water",
  "data": {
    "state": "0"
  }
}
```

## 2. **Установка порогов**

- Cистема следит за влажностью почвы
  - `меньше 40 -> почва сухая -> включить полив (water 1)`
  - `больше или равно 40 -> нормально -> ничего не делать / можно выключить полив`

- Cистема следит за температурой
  - `больше 30°C -> слишком жарко -> отправить предупреждение / что-то изменить (если есть чем управлять)`
  - `меньше или равно 30°C -> нормально -> ничего не делать`

*(Значения указаны случайным образом, необходимо согласовать.)*

```json
{
  "action": "set_thresholds",
  "data": {
    "moisture": 40,
    "temperature": 30
  }
}
```

---

## 3. Запрос данных

**ESP должна ответить**:
- влажность
- температура
- возможно свет

*Подробнее в [документации API ESP](ESPDATATOSERVER.md).*

```json
{
  "action": "get_status"
}
```

---

## 4. Отправка фото

```json
{
  "action": "upload_image",
  "data": {
    "image": "base64_string_или_url"
  }
}
```
